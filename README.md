# FPS-Game

Unity 6 tactical FPS prototype based on the included Counter-Strike style GDD.

Current foundation:

- Player movement and first-person look
- Recoil-driven hitscan weapon scaffold
- Wall penetration and surface material hooks
- Health, cash economy and round-state scaffolding
- Bomb site / bomb objective base scripts
- Offline single-player scaffolding for local player + bots, including 32v32 match bootstrap

Implementation notes are in [Docs/Advanced-CS-Clone-Roadmap.md](/E:/Unity%20Apps/repo/FPS-Game/Docs/Advanced-CS-Clone-Roadmap.md).
Offline setup steps are in [Docs/Offline-32v32-Setup.md](/E:/Unity%20Apps/repo/FPS-Game/Docs/Offline-32v32-Setup.md).
